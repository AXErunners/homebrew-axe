# DEPRECATED

> [Brew](https://brew.sh) tap for [AXE](https://github.com/axerunners/axe).

## Installation
#### Tap
```
brew tap axerunners/axe
```

#### Core client (upstream)
```
brew cask install axe-core
```
#### Thin client
```
brew cask install axe-electrum
```

## Upgrading
```
$ brew cask upgrade
```
